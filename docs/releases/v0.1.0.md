# Arch v0.1.0 Release Notes (2026-03-05)

Initial public MVP release for `arch.gocools`.

## Highlights

- AWS discovery engine for core infrastructure services.
- Versioned architecture graph schema and API.
- Terraform state import and graph-to-Terraform export.
- Guarded stack lifecycle operations with required tag enforcement.
- Drift detection and baseline policy engine integration.
- Graph diff API for deterministic change previews.
